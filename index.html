Git

Git'as yra populiariausia kodo versijavimo programa.

Git komandos

  • init - sukuria nauja git'o repozitoriją
  • add - prideda nurodytus failus prie commit'ui paruoštų failų "staged area".
  • status - nusako informaciją, apie lokalios git'o repozitorijos statusą.
  • commit -m "žinutė" - sukuria kodo versiją iš stagein'tų failų ir prideda žinutė apie pakitimus. Žinutę bandyti išlaikyti ne ilgesnę negu 50 simbolių.
  • push - iš lokalios repozitorijos naujų komitų išsiuntimas į išorinę repozitoriją.
  • pull - iš išorinės repozitorijos commit'ų atsisiuntimas į lokalią repozitoriją.
  • remote add origin https://... - išorinės repozitorijos prijungimas prie lokalios
  • log - commit'ų sąrašas
  • checkout branchName | commitID - Pereiti į kitą kodo šaką arba versiją.
  • branch naujosSakosPavadinims - sukuria naują šaką.
